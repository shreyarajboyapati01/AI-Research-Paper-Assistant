#AI Research Paper Assistant

ResearchGPT is an intelligent multi-document AI assistant that enables users to interact with PDF documents using natural language queries. The system supports semantic search, document summarization, multilingual translation, and literature review generation, making it easier to analyze and understand large collections of documents.

## 🚀 Features

- 📄 Upload and process multiple PDF documents
- 💬 Chat with documents using natural language questions
- 🔍 Semantic search using vector embeddings and FAISS
- 📝 Generate summaries from entire documents
- 🌍 Translate documents into multiple languages
- 📚 Generate literature reviews from uploaded papers
- 📥 Download summaries, translations, and literature reviews
- ⚡ FastAPI backend with Streamlit frontend

## 🛠️ Technologies Used

- Python
- FastAPI
- Streamlit
- FAISS
- Hugging Face Transformers
- Sentence Transformers
- PyPDF
- Sumy
- NLTK

## 🏗️ System Architecture

```text

    ↓

    ↓
Document Chunking

    ↓
FAISS Vector Storage
    ↓
Question Answering / Summary / Translation / Literature Review
```
